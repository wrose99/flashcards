# Music Notes Flash Cards

A focused piano note-reading practice app built with Next.js for Vercel.

## Run Locally

```bash
npm install
npm run dev
```

Open http://localhost:3000.

## Practice Controls

- Click C, D, E, F, G, A, or B to answer.
- Press those same letter keys on the keyboard to answer.
- Press Space or Enter after answering to move to the next note.
- Reset clears the current score while keeping the saved best streak.
